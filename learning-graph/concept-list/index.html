    Concept List Review Notes

    This enumeration of 200 concepts covers:

    • Foundational Geometry (Concepts 1-26): Triangle types, properties, angles, classifications
    • Pythagorean Foundation (Concepts 27-42): Theorem, triples, applications, verification
    • Core Trigonometric Ratios (Concepts 43-77): SOHCAHTOA, special angles, reciprocal functions
    • Unit Circle and Extensions (Concepts 78-107): Coordinate systems, graphs, wave characteristics
    • Identities and Formulas (Concepts 108-122): Pythagorean, sum/difference, double angle formulas
    • Triangle Solution Methods (Concepts 123-150): Laws of sines/cosines, area formulas, inverse functions
    • Real-World Applications (Concepts 151-176): Navigation, engineering, physics, architecture
    • Advanced Problem-Solving (Concepts 177-190): Equation solving, inequalities, calculator skills
    • Reference Materials (Concepts 191-200): Tables, formulas, summary guides
