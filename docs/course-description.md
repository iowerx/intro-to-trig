# Trigonometric Functions

**Title:** Trigonometric Functions

**Target Audience:** High School

**Prerequisites:** Basic geometry, basic algebra, general math and calculation skills

## Course Overview

Trigonometric Functions is a foundational course that introduces students to the relationships between angles and sides in right triangles. This course bridges algebraic thinking and geometric reasoning, providing essential tools for solving practical problems in physics, engineering, architecture, and navigation. Students will discover how the simple ratios of trigonometric functions unlock solutions to real-world challenges, from surveying land to designing structures to understanding wave behavior.

## Main Topics Covered

- Right triangles
- Pythagorean theorem
- Sine, cosine, and tangent
- Using trigonometric functions
- Finding missing lengths
- Quick reference guides and cheat sheets

## Topics Not Covered

The following topics are explicitly out of scope for this introductory course:
- Advanced trigonometric identities and equations
- Unit circle and radian measures (beyond basic introduction)
- Trigonometric functions of general angles
- Inverse trigonometric functions
- Graphs of trigonometric functions
- Applications to periodic phenomena and waves
- Polar coordinates
- Complex numbers and De Moivre's theorem
- Calculus-related applications

## Learning Outcomes

After completing this course, students will be able to:

### Remember
*Retrieving, recognizing, and recalling relevant knowledge from long-term memory.*

- Recall the definition of a right triangle
- Recall the Pythagorean Theorem
- Recall the definitions for sine, cosine, and tangent

### Understand
*Constructing meaning from instructional messages, including oral, written, and graphic communication.*

- Understand how to identify a right triangle
- Understand how to use trigonometric functions to find the length of sides that are unknown
- Explain the relationship between acute angles and trigonometric ratios in right triangles

### Apply
*Carrying out or using a procedure in a given situation.*

- Use trigonometric ratios to calculate missing sides and angles in right triangles
- Apply the Pythagorean theorem to verify trigonometric solutions
- Solve real-world problems using trigonometric functions

### Analyze
*Breaking material into constituent parts and determining how the parts relate to one another and to an overall structure or purpose.*

- Analyze what information is present in diagrams or drawings of triangles
- Determine the appropriate trigonometric ratio (sin, cos, or tan) based on given information
- Compare different trigonometric methods to determine which is most efficient for a given scenario

### Evaluate
*Making judgments based on criteria and standards through checking and critiquing.*

- Evaluate if enough information is available to use a trigonometric function
- Evaluate which trigonometric method is most efficient for a given problem

### Create
*Putting elements together to form a coherent or functional whole; reorganizing elements into a new pattern or structure.*

- Design an original real-world problem that requires trigonometric solutions
- Create a reference guide explaining when and how to apply each trigonometric function
- Develop a multi-step project involving surveying or navigation using trigonometric methods
